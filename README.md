
# React Note App

Notes Web App

-  Built with React.js and Material UI. 


## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd react-note-app
```

Install dependencies

```bash
  npm install
```

Start the server (this is an old project so npm run dev is not a script, feel free to change)

```bash
  npm run start
```
